# Life-System - v0.1

This repository contains the first versions of my website pages, built with HTML and CSS.  
Each page has its own section describing its purpose and features.

---

## Pages Overview

### 1. Login Page (`index.html`)
The Login Page allows users to enter their credentials to access the dashboard.  
**Features:**
- Input fields for Username, Gmail, and Password
- Submit button (currently placeholder, no backend)
- Centered card layout using Flexbox
- Clean modern design with shadows and rounded corners

---

### 2. Sign Up Page (`signup.html`)
The Sign Up Page lets users create a new account.  
**Features:**
- Input fields: Name, Username, Gmail, Password, Age
- Gender selection using radio buttons
- Submit button (currently placeholder)
- Placeholder texts for better UX
- Modern gradient background
- Card layout centered with Flexbox

---

## Technologies Used
- HTML5
- CSS3 (Flexbox, Shadows, Border-radius, Gradient Backgrounds)
